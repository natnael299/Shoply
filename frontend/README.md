# SHOPLY
A small scale fullstack e-commerce application done by utilizing react-js. The project has a database, an auth and a frontend where all the  infornation are diplayed in an eye-chatching way for the user.

## features
The codebase has 2 main files the frontend & the backend. the backend uses express js and has different routes for the different paths and user requests in the frontend. the frontend uses react, css and a bit of vanilla js.  the core code of the frontend is located in the src directory and in the app.jsx file all the different routes of the application are built.
